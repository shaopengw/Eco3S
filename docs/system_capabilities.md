# Eco3S 系统能力说明

## 1. 系统概述

Eco3S 是一个多智能体社会模拟系统，能够模拟居民、政府和叛军等多方主体在社会中的互动。系统支持自定义实验场景，通过配置不同的参数来研究社会动态。

## 2. 核心模块能力

### 2.1 居民系统 (Resident)
- 居民属性管理（健康指数、寿命、个性等）
- 就业状态追踪
- 满意度计算
- 社交互动
- 信息共享池

### 2.2 政府系统 (Government)
- 预算管理
- 税收调节
- 军事力量
- 政策制定
- 信息发布

### 2.3 叛军系统 (Rebels)
- 力量值管理
- 资源管理
- 策略制定
- 信息传播

### 2.4 环境系统
- 地图管理 (Map)
- 气候影响 (Climate)
- 城镇管理 (Towns)
- 就业市场 (JobMarket)
- 人口管理 (Population)
- 时间系统 (Time)
- 运输经济 (TransportEconomy)

### 2.5 社交网络 (SocialNetwork)
- 社交关系构建
- 信息传播网络
- 群体互动

## 3. 可配置参数

### 3.1 基础配置
- 初始人口数量
- 模拟总年数
- 基础生活成本
- 城镇数量及类型

### 3.2 经济参数
- 初始 GDP
- 税率范围
- 各职业基础工资
- 交通成本系数

### 3.3 社会参数
- 居民初始满意度
- 社交网络密度
- 信息传播概率
- 群体互动阈值

### 3.4 政治参数
- 政府初始预算
- 军事力量基准
- 叛军初始实力
- 政策影响系数

### 3.5 环境参数
- 气候变化幅度
- 自然资源分布
- 城镇间距离
- 环境承载能力

## 4. 输出结果

### 4.1 统计指标
- 人口变化
- GDP 增长
- 平均满意度
- 就业率
- 社会稳定性

### 4.2 事件记录
- 政策实施记录
- 叛乱事件
- 自然灾害
- 重大社会事件

### 4.3 网络分析
- 社交网络演化
- 信息传播路径
- 群体聚集度
- 意见领袖识别

## 5. API 调用方式

### 5.1 实验配置
```yaml
experiment:
  name: string            # 实验名称
  duration: int          # 模拟年数
  modules:               # 调用的模块
    - name: string       # 模块名称
    - parameters: dict   # 模块参数
  sequence: list         # 执行顺序
```

### 5.2 数据收集
```yaml
data_collection:
  metrics: list          # 需要收集的指标
  frequency: string      # 收集频率
  format: string         # 输出格式
```

### 5.3 结果分析
```yaml
analysis:
  methods: list          # 分析方法
  visualization: list    # 可视化类型
  export: dict          # 导出选项
```
