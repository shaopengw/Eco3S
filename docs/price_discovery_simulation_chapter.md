# 论文章节：基于多智能体的双市场价格发现模拟研究

## 1. 引言

在现代金融市场中，同一资产常在多个交易场所同时交易，这导致了价格发现（Price Discovery）过程的复杂性。价格发现是指市场通过买卖双方的互动，将分散、异质的信息汇集起来，最终形成一个均衡或“有效”的市场价格的过程。理解不同交易所在这一过程中的角色、效率及其相互作用，对于市场设计、监管和投资策略制定至关重要。

本研究构建了一个基于多智能体（Multi-Agent System, MAS）的模拟模型，旨在复现和探究同一只股票在两个独立交易所中的价格发现动态。通过模拟大量具有自主决策能力的股票持有者智能体在不同市场的交易行为，我们旨在回答以下核心问题：
1.  两个独立市场的价格最终会趋于一致吗？其收敛过程是怎样的？
2.  在共同价格的形成过程中，各个交易所的贡献度如何量化和评估？
3.  哪些关键因素（如市场参与者数量、交易行为模式）会影响价格发现的效率和市场的整体稳定性？

本章节将详细阐述该模拟模型的设计、核心机制、模拟流程、关键参数以及用于评估的指标体系。

## 2. 模拟模型设计

为了研究上述问题，我们设计了一个包含股票持有者智能体、交易所实体和宏观环境的计算模型。

### 2.1. 模型概述

本模型模拟了一个由众多股票持有者组成的虚拟社会。这些持有者（智能体）生活在不同的城镇中，并通过社交网络相互连接。市场上存在两个独立的交易所（交易所A和交易所B），它们同时交易同一只股票。智能体在每个模拟周期（年份）内，根据市场信息和内在逻辑做出交易决策。模型的核心是观察和记录两个交易所的价格如何因智能体的交易行为而波动，并分析它们是否以及如何收敛到一个“共同有效价格”。

### 2.2. 智能体与实体设计

#### 2.2.1. 股票持有者智能体 (Stock Holder Agent)

股票持有者是本模型的核心决策单元，具备以下特征和行为：

-   **属性**: 每个智能体拥有社会经济属性，并被分配到一个初始的城镇。他们通过社交网络（`SocialNetwork`）与其他智能体连接，但在此特定模拟中，社交网络的影响是间接的，主要体现在群体行为的宏观层面。
-   **核心行为**: 在每个模拟时间步，每个持有者独立且并行地做出交易决策。根据代码 `execute_trade` 函数的实现，该决策被简化为一个随机过程，智能体以等概率选择**购买 (buy)**、**出售 (sell)** 或 **持有 (hold)** 股票。
-   **动态演化**: 模拟引入了人口动态变化。在每个时间步开始时，系统会根据设定的出生率（`birth_rate`）生成新的股票持有者智能体（`generate_new_residents`），他们被整合进现有的城镇和居民网络中，模拟了市场参与者的持续进入。同时，智能体也存在一定的生命周期，在完成交易后会检查并处理“死亡”事件。

#### 2.2.2. 交易所实体 (Exchange Entity)

模型中包含两个固定的交易所实体，它们是市场交易的平台，而非具备自主决策能力的智能体。

-   **属性**: 每个交易所（'exchange_A', 'exchange_B'）拥有独立的**当前价格 (`current_price`)**、**交易量 (`trade_volume`)** 和**贡献度分数 (`contribution_score`)**。
-   **价格更新机制**: 交易所的价格根据其内部的交易活动进行调整。当交易发生时（买入或卖出），会产生一个微小的价格冲击（`price_impact`），驱动价格小幅随机波动。
-   **数据记录**: 交易所负责记录自身的报价历史和每个时间步的总交易量。

### 2.3. 核心机制与模拟流程

#### 2.3.1. 价格发现核心机制

价格发现是通过以下两个核心概念的计算与迭代实现的：

1.  **共同有效价格 (Common Efficient Price)**: 这是衡量整个市场价格共识的关键指标。它并非一个可直接交易的价格，而是一个计算得出的基准。根据 `calculate_common_efficient_price` 函数，其计算方式为**两个交易所当前价格的交易量加权平均值**。公式如下：
    $$ P_{common} = \frac{P_A \times V_A + P_B \times V_B}{V_A + V_B} $$
    其中，$P_A, P_B$ 分别是交易所A和B的价格，$V_A, V_B$ 是它们的交易量。如果总交易量为零，则共同有效价格为两个市场价格的简单平均。

2.  **交易所贡献度 (Exchange Contribution Score)**: 该指标用于量化每个交易所在形成共同有效价格过程中的影响力。根据 `calculate_exchange_contributions` 函数，贡献度与交易所价格和共同有效价格的偏离度成反比。价格越接近共同有效价格，贡献度越高。其计算公式为：
    $$ C_i = \frac{1}{1 + |P_i - P_{common}|} $$
    其中 $C_i$ 是交易所 $i$ 的贡献度，$P_i$ 是其当前价格。

#### 2.3.2. 模拟流程

模拟以离散的时间步（年份）进行，遵循 `simulator.run()` 方法中定义的循环。

1.  **初始化阶段**:
    -   根据 `config/price_discovery_simulation/simulation_config.yaml` 加载所有参数。
    -   创建地图、时间、人口、城镇等宏观环境对象。
    -   生成初始的股票持有者智能体群体，并构建其社交网络。
    -   初始化两个交易所，赋予其相同的 `initial_stock_price`。

2.  **迭代阶段 (按年份循环)**:
    -   **人口增长**: 根据 `birth_rate` 生成新的股票持有者智能体。
    -   **智能体行动**: 遍历所有持有者，并行执行其交易决策（买、卖或持有）。交易被随机分配到两个交易所之一。
    -   **交易所状态更新**: 交易行为触发对应交易所的交易量累加和价格微调。
    -   **核心指标计算**: 在所有交易完成后，计算当前年份的“共同有效价格”和各交易所的“贡献度”。
    -   **数据收集**: 调用 `collect_results` 方法，记录当前年份的所有关键指标（详见第4节）。
    -   **时间推进**: 时间进入下一年，循环继续，直至达到 `total_years`。

3.  **结束与分析阶段**:
    -   模拟循环结束后，调用 `analyze_price_trends` 方法，对收集到的时间序列数据进行统计分析（如计算均值、标准差等）。
    -   调用 `save_results` 将所有原始数据保存到 CSV 文件中。
    -   调用 `plot_all_results` 生成一系列可视化图表并保存，用于直观分析。

## 3. 关键参数与可配置性

本模型具有高度的可配置性，研究者可通过修改位于 `config/price_discovery_simulation/` 目录下的 `simulation_config.yaml` 文件来调整实验条件。主要参数包括：

-   `initial_stock_price`: 股票在所有交易所的起始价格。
-   `start_year` / `total_years`: 模拟的起止时间范围。
-   `initial_population`: 初始市场参与者（股票持有者）的数量。
-   `birth_rate`: 人口年增长率，用于模拟新参与者的进入速度。
-   `trade_frequency`: 交易频率（当前固定为每年一次）。
-   `price_update_interval`: 价格更新间隔（当前固定为每年一次）。

通过调整这些参数，可以系统性地研究市场规模、参与者增长速度等因素对价格发现过程的影响。

## 4. 数据收集与评估指标

为了全面评估价格发现过程的动态和效率，我们在每个模拟年份收集了以下一系列关键指标：

| 指标名称 (代码内) | 指标含义与计算方式 | 研究意义 |
| :--- | :--- | :--- |
| `exchange_A_price` / `exchange_B_price` | 交易所A和B各自的股票价格。 | 跟踪和比较不同市场的价格走势，是分析价格分歧与收敛的基础。 |
| `common_efficient_price` | 两个交易所的交易量加权平均价格。 | 代表了市场的整体价格共识，是评估市场稳定性和效率的核心基准。 |
| `exchange_A_volume` / `exchange_B_volume` | 交易所A和B各自的成交量。 | 反映了市场的流动性分布和交易活跃度，是计算共同有效价格的权重。 |
| `exchange_A_contribution` / `exchange_B_contribution` | 交易所对共同有效价格的贡献度分数。 | 量化了每个市场在价格发现过程中的影响力。 |
| `price_discovery_efficiency` | 价格发现效率，为所有交易所贡献度的算术平均值。 | 衡量市场将信息整合到价格中的整体速度和有效性。 |
| `price_variance` | 两个交易所价格与共同有效价格之间的方差。 | 量化了市场间价格的分歧程度，是衡量市场割裂的指标。 |
| `price_stability` | 价格稳定性，计算为 `1 / (1 + price_variance)`。 | 评估市场价格的波动风险。方差越小，稳定性越高。 |
| `price_convergence_rate` | 价格收敛率，衡量共同有效价格在相邻时间步的变化。 | 反映了市场价格共识形成的速度。 |
| `average_price_deviation` | 各交易所价格与共同有效价格的平均绝对偏差。 | 直观地反映了两个市场的价格偏离程度，是衡量价格收敛性的直接指标。 |
| `cumulative_efficiency` | 累积价格发现效率，为历史效率的平均值。 | 评估模拟全程的平均效率表现，平滑短期波动。 |
| `population` | 股票持有者总数。 | 跟踪市场参与者规模的变化，可与其它指标关联分析规模效应。 |

## 5. 结果呈现与分析

模拟的输出结果是多维度的，为深入分析提供了坚实基础：

1.  **原始数据**: 所有在模拟过程中收集的指标数据，以时间序列的形式保存在 `data/` 目录下的 CSV 文件中。这为精细的量化分析、统计检验和计量经济学建模提供了可能。
2.  **可视化图表**: 模拟自动生成一系列图表，保存在 `experiment_dataset/price_discovery_simulation/plots/` 目录下，包括：
    -   **价格走势图**: 将两个交易所的价格与共同有效价格绘制在同一张图上，直观展示价格的收敛或发散趋势。
    -   **交易量对比图**: 以柱状图形式比较两个交易所的交易量，分析流动性的分布。
    -   **贡献度趋势图**: 展示两个交易所贡献度随时间的变化，分析市场影响力的演变。
    -   **效率与稳定性图**: 分别绘制价格发现效率和价格稳定性的时间序列，评估市场的宏观表现。

通过对这些结果的综合分析，研究者可以验证关于价格收敛的假设，识别在价格发现中起主导作用的市场，并探索不同参数设置对市场整体效率和稳定性的深层影响。
