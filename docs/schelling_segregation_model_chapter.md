# 论文章节：基于多智能体的谢林空间隔离模拟研究

## 1. 引言

城市空间隔离（Spatial Segregation）是社会学和城市研究领域中的一个核心问题。诺贝尔经济学奖得主托马斯·谢林（Thomas Schelling）于1971年提出的隔离模型，揭示了一个令人深思的现象：即使个体对邻居的族群偏好相对温和，也可能在宏观层面涌现出高度的空间隔离格局。这一发现对理解城市族群分布、社区形成以及社会融合具有深远意义。

本研究基于多智能体系统（Multi-Agent System, MAS）重构了谢林隔离模型，旨在通过计算模拟深入探究以下核心问题：
1. 在不同的聚居倾向阈值下，城市空间隔离格局如何演化？
2. 居民的微观迁移决策如何在宏观层面导致族群聚集和隔离现象？
3. 哪些关键因素（如相似度阈值、人口密度、迁移成本）会影响隔离模式的形成速度和稳定性？

本章节将系统阐述该模拟模型的设计原理、核心机制、实施流程、关键参数配置以及多维度的评估指标体系。

## 2. 模拟模型设计

### 2.1. 模型概述

本模型构建了一个由大量具有自主决策能力的居民智能体组成的虚拟城市系统。居民被分为两种类型（类型A和类型B），他们分布在一个二维网格地图上，每个位置代表一个可居住的空间单元。智能体根据其邻居的族群构成做出满意度评估和迁移决策。模型的核心目标是观察和量化：在个体层面的温和偏好下，宏观层面的空间隔离格局如何自发涌现。

### 2.2. 智能体与实体设计

#### 2.2.1. 居民智能体 (Resident Agent)

居民智能体是本模型的核心决策单元，具备以下特征和行为机制：

-   **族群属性**: 每个居民被随机分配为类型A或类型B（`resident_type`），该属性在模拟过程中保持不变，是影响其偏好和决策的关键因素。
-   **空间属性**: 每个居民占据地图上的一个位置（`location`），通过坐标 $(x, y)$ 表示。居民可以感知其 Moore 邻域（周围8个格子）内其他居民的族群类型。
-   **经济属性**: 居民拥有收入、满意度（`satisfaction`）和健康指数等社会经济属性，这些属性受到邻居相似度和基本生活成本的影响。
-   **核心行为**: 
    - **邻居相似度感知**: 居民通过 `calculate_neighbor_similarity` 函数计算其邻域内同族群居民的比例。
    - **迁移决策**: 在每个时间步，居民基于邻居相似度与内在阈值（`similarity_threshold`）的比较，通过 LLM（大语言模型）辅助决策是否迁移到新位置。当邻居相似度低于阈值时，居民倾向于迁移。
    - **决策执行**: 通过 `execute_decision` 方法，居民在地图上寻找满意的新位置并完成迁移。
-   **生命周期**: 模拟引入了人口动态变化机制，包括新居民的出生（`generate_new_residents`）和现有居民的死亡（基于健康指数和生活成本），模拟了真实的人口流动。

#### 2.2.2. 环境实体

模型包含以下支撑性环境实体：

-   **地图 (Map)**: 一个 $W \times H$ 的二维网格空间，定义了居民的可居住区域。地图的宽度和高度可通过配置文件调整。
-   **城镇 (Towns)**: 地图被划分为多个城镇区域，每个城镇管理其范围内的居民群体（`resident_group`），并提供就业市场等服务。
-   **社交网络 (Social Network)**: 通过图结构连接居民，支持信息传播和社会影响的模拟。网络定期更新以反映人口变化和迁移。
-   **时间 (Time)**: 离散时间系统，以年份为单位推进模拟，控制模拟的总时长。

### 2.3. 核心机制与模拟流程

#### 2.3.1. 空间隔离核心机制

模型的空间隔离涌现基于以下三个核心计算机制：

1.  **邻居相似度 (Neighbor Similarity)**: 这是驱动居民决策的关键指标。对于居民 $i$ 在位置 $(x, y)$，其邻居相似度 $S_i$ 的计算方式为：
    $$ S_i = \frac{N_{same}}{N_{total}} $$
    其中，$N_{same}$ 是邻域内与居民 $i$ 同类型的邻居数量，$N_{total}$ 是邻域内的总邻居数量。如果邻域为空，则 $S_i = 0$。

2.  **相似度阈值 (Similarity Threshold)**: 这是居民的内在偏好参数 $\theta$，表示其期望的最低邻居相似度。当 $S_i < \theta$ 时，居民产生迁移倾向。本模型支持配置不同的阈值水平（如30%、50%、80%），以研究偏好强度对隔离的影响。

3.  **隔离指数 (Segregation Index)**: 这是宏观层面衡量空间隔离程度的指标。通过计算类型A和类型B居民聚集区域之间的最小空间距离 $D_{min}$，并归一化到最大可能距离：
    $$ I_{seg} = \frac{D_{min}}{D_{max}} $$
    其中 $D_{max} = W + H$ 为地图的最大曼哈顿距离。$I_{seg}$ 越大，表示两个族群的空间隔离程度越高。

#### 2.3.2. 模拟流程

模拟遵循 `simulator.run()` 方法中定义的迭代流程，以离散的时间步（年份）推进：

1.  **初始化阶段**:
    -   根据 `config/schelling_segregation_model/simulation_config.yaml` 加载所有参数。
    -   创建地图、时间、人口、城镇等环境对象。
    -   生成初始的居民智能体群体（`generate_canal_agents`），并为每个居民随机分配类型A或B。
    -   构建社交网络，连接居民智能体。

2.  **迭代阶段 (按年份循环)**:
    -   **状态更新**: 
        - 计算平均满意度（`calculate_average_satisfaction`），并根据满意度动态调整人口出生率。
    -   **人口增长**: 
        - 根据当前出生率生成新的居民智能体，随机分配类型并整合到现有系统中（`integrate_new_residents`）。
    -   **智能体决策与行动**: 
        - 遍历所有居民，计算每个居民的邻居相似度。
        - 调用 `decide_action_by_llm` 方法，居民基于相似度和阈值通过LLM做出迁移或留守决策。
        - 并行执行所有决策（使用 `asyncio.gather`），记录迁移行为。
    -   **生命周期更新**: 
        - 检查居民的健康状态和生存条件，处理死亡事件并更新人口统计。
    -   **网络维护**: 
        - 每隔3-5年随机更新社交网络边，模拟社会关系的动态变化。
    -   **数据收集**: 
        - 调用 `collect_results` 方法，记录当前年份的所有关键指标（详见第4节）。
    -   **结果保存**: 
        - 将当前时间步的数据追加保存到CSV文件中。
    -   **时间推进**: 
        - 时间进入下一年，循环继续，直至达到 `total_years`。

3.  **结束与分析阶段**:
    -   模拟循环结束后，完整保存所有时间序列数据。
    -   调用可视化函数（`visualize_results`）生成多维度图表，包括迁移率、隔离指数、族群分布等时间序列曲线。
    -   输出模拟总耗时和最终统计摘要。

## 3. 关键参数与可配置性

本模型具有高度的可配置性，研究者可通过修改位于 `config/schelling_segregation_model/` 目录下的 `simulation_config.yaml` 文件来系统性地调整实验条件。主要参数包括：

### 3.1. 基础模拟参数

-   `initial_population`: 初始居民数量，决定模拟的规模和密度。
-   `start_year` / `total_years`: 模拟的起止时间范围，控制观察隔离涌现的时间窗口。
-   `map_width` / `map_height`: 地图的宽度和高度，定义了空间网格的规模。
-   `birth_rate`: 人口年增长率基准，实际出生率会根据平均满意度动态调整。

### 3.2. 核心行为参数

-   `similarity_threshold`: 居民的相似度阈值，是控制隔离强度的核心参数。配置文件支持定义多个实验场景：
    - `baseline_50percent`: 基线场景，阈值为 0.5（50%）
    - `low_preference_30percent`: 低偏好场景，阈值为 0.3（30%）
    - `high_preference_80percent`: 高偏好场景，阈值为 0.8（80%）
-   `basic_living_cost`: 基本生活成本，影响居民的满意度和生存状态。

### 3.3. 实验控制参数

-   `active_scenario`: 当前激活的实验场景名称。
-   `resume_from_cache`: 是否从缓存恢复模拟，支持中断后的续跑。

通过系统性地调整这些参数，研究者可以探索不同偏好强度、人口密度和空间约束条件下的隔离动态，验证谢林模型的理论预测，并发现可能的非线性临界效应。

## 4. 数据收集与评估指标

为了全面评估空间隔离的动态过程和影响因素，模拟在每个时间步收集以下多维度指标：

| 指标名称 (代码内) | 指标含义与计算方式 | 研究意义 |
| :--- | :--- | :--- |
| `years` | 当前模拟年份 | 时间序列的横轴，用于跟踪所有指标的演化过程。 |
| `population` | 居民总数 | 反映人口规模的变化，是计算各类比率指标的分母。 |
| `migration_rate` | 迁移率，当前年份迁移人数占总人口的百分比 | 直接衡量居民流动的活跃程度，是隔离动态的直接驱动力。高迁移率表明系统尚未达到稳态。 |
| `average_satisfaction` | 平均满意度，所有居民满意度的算术平均值 | 反映居民对当前居住环境的整体感受，是影响出生率和迁移决策的重要因素。 |
| `average_similarity` | 平均相似度，所有居民邻居相似度的算术平均值 | 宏观层面衡量族群聚集程度，是隔离涌现的微观基础。该值越接近1，表示"物以类聚"现象越明显。 |
| `segregation_index` | 隔离指数，基于类型A和类型B聚集区域间最小距离的归一化指标 | 量化空间隔离的核心指标，直观反映两个族群在地理上的分离程度。 |
| `type_a_population` | 类型A居民数量 | 跟踪类型A族群的人口规模变化，分析族群间是否存在增长差异。 |
| `type_b_population` | 类型B居民数量 | 跟踪类型B族群的人口规模变化，与type_a_population配合分析族群比例的动态。 |

### 4.1. 指标间的逻辑关系

这些指标构成了一个完整的评估体系，相互关联：
-   **微观行为 → 宏观格局**: `migration_rate` 反映微观决策的活跃度，`average_similarity` 和 `segregation_index` 体现宏观隔离格局。
-   **动态演化**: 通过观察 `migration_rate` 从高到低的变化，可以识别系统达到稳态的时间点。
-   **均衡状态**: 当 `migration_rate` 趋近于零，且 `average_similarity` 和 `segregation_index` 趋于稳定时，表明系统达到了准平衡状态。

## 5. 结果呈现与分析

模拟输出的结果具有多层次性，支持定量分析和直观可视化：

### 5.1. 原始数据

所有在模拟过程中收集的指标数据以时间序列的形式保存在 `experiment_dataset/schelling_segregation_model/data/` 目录下的CSV文件中。文件命名格式为 `running_data_<timestamp>_pid<process_id>.csv`，确保了并行实验的数据隔离。这些原始数据为精细的统计分析、假设检验和计量建模提供了基础。

### 5.2. 可视化图表

模拟自动生成一系列时间序列图表，保存在 `experiment_dataset/schelling_segregation_model/plots/` 目录下，包括：

1.  **人口演化图 (`population_*.png`)**: 
    -   展示总人口随时间的变化趋势，验证人口增长机制的有效性。
    
2.  **迁移率时间序列图 (`migration_rate_*.png`)**: 
    -   绘制迁移率的时间曲线，是识别系统稳态的关键图表。典型模式为：初期高迁移率，随着居民找到满意位置而逐渐下降，最终趋于稳定。
    
3.  **平均满意度趋势图 (`satisfaction_*.png`)**: 
    -   跟踪居民满意度的演化，与迁移率互补，共同解释居民的行为动机。
    
4.  **平均相似度时间序列图 (`average_similarity_*.png`)**: 
    -   展示邻居相似度如何随时间增长，直观反映"物以类聚"现象的涌现过程。在基线场景（阈值50%）下，预期该值会从初始的随机水平（约0.5）逐渐上升到更高水平。
    
5.  **隔离指数演化图 (`segregation_index_*.png`)**: 
    -   量化展示空间隔离程度的增长过程，是验证谢林理论的核心图表。当阈值较低（如30%）时，隔离仍可能涌现；当阈值较高（如80%）时，隔离会更快速和显著。
    
6.  **族群分布对比图 (`resident_types_*.png`)**: 
    -   对比类型A和类型B的人口数量随时间的变化，分析是否存在族群间的增长不平衡。

### 5.3. 结果分析框架

通过对这些结果的综合分析，研究者可以：

1.  **验证理论假设**: 检验即使在温和偏好（如50%阈值）下，宏观隔离是否仍会涌现，验证谢林的核心论断。
   
2.  **参数敏感性分析**: 对比不同 `similarity_threshold` 场景下的结果，量化偏好强度对隔离速度和最终程度的影响。
   
3.  **动态路径分析**: 追踪从初始随机分布到稳态隔离的演化路径，识别关键的临界点和相变现象。
   
4.  **政策启示**: 通过调整基本生活成本、迁移成本等参数，探索可能的政策干预措施对减缓隔离的效果。

## 6. 结论与展望

本研究通过多智能体模拟重构了谢林隔离模型，提供了一个高度可配置且可扩展的计算实验平台。模拟结果不仅能够重现经典理论的核心发现，还能够支持更丰富的场景探索和机制分析。未来的研究方向包括：
-   引入异质性：考虑居民收入、教育等社会经济属性的差异。
-   多族群扩展：将模型从两类族群扩展到多类族群，研究更复杂的隔离模式。
-   网络效应：深化社交网络对迁移决策的影响机制。
-   政策实验：模拟混合居住政策、公共住房分配等干预措施的效果。

通过这些扩展，该模拟平台有望成为城市规划、社会政策研究和复杂系统科学教学的有力工具。
