# 多智能体模拟实验设计文档

## 1. 基本信息

**实验名称：** customer_satisfaction_sim

**实验目标：** 模拟观察服务质量和产品体验动态变化对顾客满意度、忠诚度和购买决策的影响

**背景说明：** 本实验旨在研究服务质量和产品体验对顾客行为的影响，以探讨提升顾客满意度和忠诚度的策略。
