# TEOG 模拟——两河流域治理与民生场景说明

## 1. 基本信息

**实验名称：** TEOG

**实验目标：** 研究两河流域城市带中人口、就业、政府治理与交通经济的动态演化，以及河运通航对社会满意度与经济活力的影响。

**背景说明：** 本模拟以底格里斯河与幼发拉底河沿线城市为地理骨架，居民在真实坐标的城镇网络中生活、就业与互动。政府通过财政与治理手段影响民生和河运通航，进而作用于居民满意度与经济表现。

## 2. 系统模块
- **居民系统** - 个体智能体的行为决策
- **地图系统** - 两河流域城镇分布与空间计算
- **时间系统** - 模拟时间步进管理
- **人口系统** - 人口增长与规模变化
- **政府系统** - 政策与公共预算管理
- **就业市场** - 职业选择、工资体系与就业分配
- **社交网络** - 人际关系与信息传播
- **运输经济** - 河运通航与运输成本
- **气候系统** - 气候变化对通航与经济的影响

## 3. 智能体设计

### 居民智能体
- **数量：** 初始10人（可随出生率增长）
- **关键属性：**
	- 满意度（satisfaction）- 反映生活满意程度
	- 收入（income）- 来自职业的年收入
	- 职业（job）- 农民 / 城市居民
	- 位置（town）- 所在城镇
	- 社交网络连接 - 与其他居民的社交关系
- **核心行为（已实现）：**
	- **就业相关：**寻找工作或继续工作
	- **社交互动：**与邻居交流、受信息影响

### 政府智能体
- **组成：** 多个政府官员（OrdinaryGovernmentAgent）+ 高级官员（HighRankingGovernmentAgent）
- **核心职能：**
	- 财政预算分配与政策执行
	- 河运通航维护投入
	- 社会治理与公共服务
- **关键属性：**
	- 预算（budget）
	- 税率（tax_rate）
	- 军事力量（military_strength）
- **决策机制：** 支持群体决策（启用时进行2轮讨论）

## 4. 环境设置

### 4.1 地理环境
- **城镇数量：** 18个（15个河道沿线城镇 + 3个其他城镇）
- **空间特征：**
	- 底格里斯河与幼发拉底河两条水系
	- 城镇沿河分布，河运通航连接经济活动
	- 地理范围：经度43.0°-48.0°，纬度29.0°-36.0°
- **真实性：** 真实地理坐标（`is_real_world: true`）

### 4.2 气候条件
- **变化模式：** 读取气候数据（experiment_dataset/climate_data/climate.csv）
- **影响路径：** 气候变化影响河流通航能力，进而影响运输成本与经济活力

### 4.3 经济环境
- **基础职业与工资：**
	- 农民：10两/年
	- 城市居民：20两/年
- **GDP计算：** 所有非政府主体的收入总和

## 5. 关键指标

**模拟过程中监测的核心指标及其意义：**

- **人口总量 (Population)** - 反映社会规模变化
- **平均满意度 (Average Satisfaction)** - 反映民生与治理效果
- **政府预算 (Government Budget)** - 反映政府财政能力
- **税率 (Tax Rate)** - 影响政府收入与居民可支配收入
- **运河通航能力 (River Navigability)** - 反映河运状态
- **GDP (Gross Domestic Product)** - 反映经济活力
- **城市规模 (Urban Scale)** - 反映城市集聚程度

## 6. 模拟参数

### 6.1 基础参数
- **初始人口数量 (initial_population)：** 10人
- **模拟起始年份 (start_year)：** 1650年
- **模拟总年数 (total_years)：** 2年
- **出生率 (birth_rate)：** 0.05
- **地图宽度 (map_width)：** 100
- **地图高度 (map_height)：** 150
- **模拟名称 (simulation_name)：** 未指定（默认自动生成）

### 6.2 行为参数
- **响应概率 (response_probability)：** 0.05
- **最大重试次数 (max_retry_attempts)：** 3
- **重试延迟 (retry_delay)：** 1秒
- **是否从缓存恢复 (resume_from_cache)：** 0
- **是否保存缓存 (save_cache)：** 1（启用）

### 6.3 群体决策参数
- **政府群体决策 (group_decision.government)：**
	- **启用状态 (enabled)：** 1
	- **最大讨论轮数 (max_rounds)：** 2轮

## 7. 交互机制

**智能体之间的关键交互流程：**

- **居民 ↔ 政府：**
	- 政府政策与预算分配影响居民满意度与就业
	- 税率调整影响居民收入与整体经济表现

- **居民 ↔ 居民：**
	- 社交网络传播信息与观点

- **居民 ↔ 环境：**
	- 河运通航变化影响就业与经济机会
	- 城镇区位影响迁移与资源获取

## 8. 预期结果

**研究假设：**
1. 河运通航能力提升会提高经济活动与居民满意度
2. 政府预算分配策略对社会稳定与经济表现具有显著影响
3. 城镇网络结构会影响资源在空间上的扩散效率

**预期发现：**
- 河运维护与经济活力之间存在正向反馈
- 政策波动会在短期内造成满意度起伏
- 城镇节点的区位优势会导致发展不均衡
