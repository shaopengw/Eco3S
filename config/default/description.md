# AgentWorld 默认模拟场景说明

本模拟以清代运河沿线社会为背景，聚焦于人口、经济、政府、叛军、社交网络等要素的动态演化。模拟过程中，居民、政府与叛军等多方智能体在地理、经济、社会和气候等环境影响下互动，展现社会系统的复杂变化。

## 主要内容
居民根据生存需求、职业选择和社交网络进行行为决策，并对政府满意度产生反馈。
政府通过税收、预算管理、运河维护和社会治理等措施影响社会稳定与经济发展。
叛军可发起叛乱，影响社会秩序，其力量和资源随局势变化。
社交网络动态演化，影响信息传播与群体行为。
气候与运河通航状况影响经济活动和社会运行。
系统支持群体决策机制，模拟政府和叛军内部的集体决策过程。

## 关键指标
- 人口数量
- 失业率
- 政府预算
- 叛军力量
- 居民满意度
- 税率
- 运河通航性
- GDP

## 数据记录
系统会记录并可视化上述关键指标随时间的变化，保存在指定的数据目录中。可以通过这些数据分析社会系统的演化过程。
