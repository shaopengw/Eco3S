# Eco3S 默认模拟场景说明

## 1. 基本信息

**实验名称：** default

**实验目标：** 研究清代运河沿线社会中人口、经济、政府、叛军等要素的动态演化及其相互作用机制

**背景说明：** 本模拟以清代运河沿线社会为背景，重点关注京杭大运河及其沿线15个主要城镇和12个非运河城镇的社会经济系统。模拟期间（1650-1665年），居民、政府与叛军等多方智能体在地理、经济、社会和气候等环境影响下进行互动，展现社会系统的复杂变化规律。

## 2. 系统模块
- **居民系统** - 个体智能体的行为决策
- **地图系统** - 地理空间和城镇分布
- **时间系统** - 模拟时间步进管理
- **人口系统** - 人口增长、迁移、死亡
- **政府系统** - 政策制定、税收管理、军事力量维护
- **叛军系统** - 社会冲突、叛乱模拟、资源掠夺
- **气候系统** - 天气变化、自然灾害对运河和经济的影响
- **就业市场** - 职业选择、失业率、工资体系
- **社交网络** - 人际关系、信息传播、群体行为
- **运输经济** - 运河贸易、运输成本、经济活动

## 3. 智能体设计

### 居民智能体
- **数量：** 初始2000人，根据出生率动态增长
- **关键属性：** 
  - 满意度（satisfaction）- 反映对生活和政府的满意程度
  - 收入（income）- 来自职业的年收入
  - 职业（job）- 当前从事的工作类型
  - 位置（town）- 所在城镇
  - 社交网络连接 - 与其他居民的社交关系
- **核心行为（已实现）：**
  - **就业相关：**
    - `work` - 寻找工作或继续工作，可指定期望职业和最低薪资
  - **迁移相关：**
    - `migrate` - 迁移到新城镇，可选择是否更新职业
  - **社会冲突：**
    - `join_rebellion` - 参加叛军（需满足一定条件）
  - **社交互动：**
    - 与邻居进行社交对话
    - 受叛军宣传影响
    - 在社交网络中传播信息

### 政府智能体
- **组成：** 多个政府官员（OrdinaryGovernmentAgent）+ 高级官员（HighRankingGovernmentAgent）+ 信息整理员（InformationOfficer）
- **核心职能：** 
  - 税收政策制定与调整
  - 运河维护资源分配
  - 军事力量部署与叛乱镇压
  - 社会治理与民生改善
- **关键属性：** 
  - 预算（budget）- 政府可用资金
  - 军事力量（military_strength）- 官员及士兵总数
  - 税率（tax_rate）- 当前税收比例
- **决策机制：** 支持群体决策（启用时进行2轮讨论，由信息整理员汇总，领导决策）

### 叛军智能体
- **组成：** 多个普通叛军（OrdinaryRebel）+ 叛军领袖（RebelLeader）+ 信息整理员（RebelsInformationOfficer）
- **核心职能：** 
  - 发动叛乱攻击城镇
  - 宣传煽动居民加入
  - 掠夺资源壮大力量
  - 挑战政府权威
- **关键属性：** 
  - 力量（strength）- 叛军总人数
  - 资源（resources）- 可用资金
  - 士气 - 影响叛乱成功率
- **决策机制：** 支持群体决策（当前配置禁用，由领袖直接决策）

## 4. 环境设置

### 4.1 地理环境
- **城镇数量：** 27个（15个运河沿线城镇 + 12个非运河城镇）
- **空间特征：** 
  - 运河沿线城镇：沿京杭大运河分布，从北京到杭州
  - 非运河城镇：分散分布于运河周边地区
  - 地理范围：经度109.0°-125.0°，纬度30.0°-41.0°
- **自然资源分布：** 运河沿线城镇具有运河贸易优势，非运河城镇以农业为主
- **城镇间距离：** 基于真实地理坐标计算，影响迁移成本和运输成本

### 4.2 气候条件
- **变化模式：** 基于历史气候数据（experiment_dataset/climate_data/climate.csv），模拟气候变化对运河通航的影响
- **极端事件：** 气候异常导致运河通航能力下降，影响贸易和就业
- **气候变化幅度：** 气候影响因子随时间动态变化，影响运河自然衰减速率

### 4.3 经济环境
- **基础生活成本：** 8两/年，居民维持基本生存所需
- **GDP计算：** 所有非叛军居民的收入总和

### 4.4 职业与就业市场
- **职业类型与基础工资：**
  - 农民：10两/年
  - 普通工作者：12两/年
  - 运河维护工：15两/年
  - 官员及士兵：20两/年
  - 商人：30两/年
  - 叛军：8两/年
  
- **职业分布比例（按城镇类型）：**
  - 沿河城镇：商业和运河维护工比例较高（商人5-10%，运河维护工20-30%）
  - 非沿河城镇：农民比例较高（70-80%），无运河维护工

## 5. 关键指标

**模拟过程中监测的核心指标及其意义：**

- **人口总量 (Population)** - 反映社会规模和人口增长趋势，受出生率和死亡率影响
- **平均满意度 (Average Satisfaction)** - 居民整体满意程度（0-100），反映社会和谐度和政府治理效果
- **GDP (Gross Domestic Product)** - 所有非叛军居民的年收入总和，反映经济发展水平
- **失业率 (Unemployment Rate)** - 未就业居民占总人口的比例，反映就业市场状况
- **政府预算 (Government Budget)** - 政府可用资金，影响政策执行能力
- **叛军力量 (Rebellion Strength)** - 叛军总人数，反映社会冲突强度
- **叛军资源 (Rebellion Resources)** - 叛军可用资金，影响叛乱能力
- **税率 (Tax Rate)** - 当前税收比例，影响政府收入和居民满意度
- **运河通航能力 (River Navigability)** - 运河状态（0-100），影响运河相关职业就业和贸易
- **叛乱次数 (Rebellions)** - 每年发生的叛乱事件数量，反映社会稳定性

## 6. 模拟参数

### 6.1 基础参数
- **初始人口数量 (initial_population)：** 2000人
- **模拟起始年份 (start_year)：** 1650年
- **模拟总年数 (total_years)：** 15年（1650-1665）
- **出生率 (birth_rate)：** 0.05（基准值，随平均满意度动态调整）
- **地图宽度 (map_width)：** 100
- **地图高度 (map_height)：** 150
- **模拟名称 (simulation_name)：** ""（默认为空，可自定义）

### 6.2 行为参数
- **响应概率 (response_probability)：** 0.05（居民每年执行行为决策的概率）
- **最大重试次数 (max_retry_attempts)：** 3（LLM调用失败时的重试次数）
- **重试延迟 (retry_delay)：** 1秒
- **是否从缓存恢复 (resume_from_cache)：** 0（禁用）
- **是否保存缓存 (save_cache)：** 0（禁用）

### 6.3 群体决策参数
- **政府群体决策 (group_decision.government)：**
  - **启用状态 (enabled)：** 1（启用）
  - **最大讨论轮数 (max_rounds)：** 2轮
- **叛军群体决策 (group_decision.rebellion)：**
  - **启用状态 (enabled)：** 0（禁用，由叛军领袖直接决策）
  - **最大讨论轮数 (max_rounds)：** 2轮

## 7. 交互机制

**智能体之间的关键交互流程：**

- **居民 ↔ 政府：** 
  - 居民通过满意度反馈影响政府决策
  - 政府税率调整影响居民收入和满意度
  - 政府军事力量与叛军作战，影响居民安全感
  - 政府运河维护投入影响居民就业机会

- **居民 ↔ 居民：** 
  - 社交网络中的信息传播（每3-5年更新一次网络结构）
  - 叛军宣传影响邻居加入叛乱
  - 社交对话影响观点和行为决策

- **居民 ↔ 环境：** 
  - 气候变化影响运河通航，进而影响就业选择
  - 运河状态影响运河维护工和商人的工作机会
  - 居民根据城镇经济状况决定是否迁移

- **政府 ↔ 叛军：** 
  - 叛军发起叛乱攻击城镇，政府派军队镇压
  - 基于非线性优势损耗模型计算战斗结果
  - 双方力量对比影响战斗损耗和叛乱成功率

- **叛军 ↔ 居民：**
  - 叛军通过宣传煽动居民加入
  - 居民满意度低时更容易被煽动
  - 叛军从支持者中获取资源

## 8. 预期结果

**研究假设：**
1. 政府税率与居民满意度存在非线性关系，过高税率会导致满意度下降和叛乱增加
2. 运河通航能力的变化会显著影响沿河城镇的就业率和经济发展
3. 群体决策机制（政府启用）会产生更稳健的政策，相比单一领导决策能更好地平衡多方利益

**预期发现：**
- 社会系统的稳定性受到经济、政治、气候等多因素的共同影响
- 运河作为关键基础设施，其维护投入与经济发展呈现正相关关系
- 叛军力量在特定条件下（低满意度、高失业率）会快速增长
- 政府预算管理策略对长期社会稳定具有重要影响

## 补充说明

**特殊设定：**
- 叛军收入固定为6两/人/年（基于力量值计算）
- 战斗损耗采用非线性优势损耗模型，考虑极端优势保护和随机扰动
- 社交网络每3-5年随机更新一次结构
- 居民出生时自动加入社交网络并分配到城镇

**已知限制：**
- 模拟时间较短（15年），可能无法观察到某些长期趋势
- 居民决策基于LLM生成，存在一定随机性
- 气候数据基于历史记录，未考虑突发性极端气候事件
- 职业工资为固定值，未考虑市场供需动态调整
