# 决策型（decision）模拟实验设计模板

> 写作风格：尽量用自然语言叙述，避免“配置表/模块清单式”的堆砌；但必须保留清晰的指标口径，便于后续对比实验与复现。

## 1. 基本信息

本场景用于研究：【一句话说明要研究的核心问题与因果链路，例如“外部冲击 → 经济/就业变化 → 迁移/冲突 → 政府政策反馈”】【要求：不要写成条目清单】

背景设定：【2–4 句话说明时空背景、社会结构或系统边界，说明“谁在什么环境下做什么决策”】【要求：尽量具体】

实验窗口与步进：【例如“1650–1665 年，以年为步进”或“共 N 年/轮”】【要求：写出开始、结束/总步数】

初始规模与约束：【例如初始人口、城镇数量、基本生活成本、关键资源/预算约束等。若支持缓存恢复/保存，请说明“由运行配置控制”】【要求：只保留必要口径】

原需求（尽可能保留）：
【粘贴用户原始需求，保留关键条件、阈值、公式与名词。】

## 2. 智能体与行为

居民/个体主体：【说明数量级、关键状态（如收入、满意度、位置/城镇、职业）、以及会做出的关键决策（就业、迁移、加入/退出组织、对话/传播等）。强调“哪些决策会改变哪些状态”。】

政府/组织主体（如有）：【说明它的目标与约束（预算、税率、治理能力等），以及它通常如何做决策（单人/群体决策、讨论轮数上限等），并说明它会如何反馈到居民与环境。】

对立/冲突主体（如有）：【说明其扩张/动员方式、资源来源与关键状态（力量、资源、士气/成功率等），以及与政府/居民的交互。】

## 3. 环境与经济设定

空间结构：【城镇/区域的数量、类型差异（如沿河/非沿河、城市/乡村），以及距离/交通对迁移与贸易的作用。】

外部过程：【气候/灾害/政策等外生过程的来源与作用方式，尤其是它如何影响基础设施状态（如通航能力）与机会结构（岗位、贸易、成本）。】

经济口径：【基础生活成本、收入/工资的基本口径；GDP 或其他宏观指标如何从微观汇总得到（用一句话说明即可）。】

## 4. 交互机制

用 1 段话说明最重要的闭环（建议写成“因素 A → 机制 B → 结果 C → 反馈 D”），并点明：
- 哪些变量是外生冲击（实验控制变量），哪些是系统响应（被观察变量）；
- 哪些机制会导致非线性/阈值效应（例如税率-满意度、通航-就业、战斗损耗等）。

## 5. 关键指标

- 人口总量（Population）：建议记录总人口、各城镇人口、净迁移量（迁入-迁出）。
- 平均满意度（Average Satisfaction）：建议记录均值，必要时加分位数/方差以捕捉“不满集中化”。
- GDP（Gross Domestic Product）：按场景口径汇总（例如非叛军居民收入总和），必要时分组对比（如沿河/非沿河）。
- 失业率（Unemployment Rate）：建议记录全局与各城镇失业率，并保留职业分布摘要用于解释。
- 政府预算（Government Budget）：建议记录预算余额及年度变化（若可分解收入/支出更好）。
- 税率（Tax Rate）：作为关键政策变量，建议与满意度/稳定性指标做对照分析。
- 基础设施状态（如 River Navigability，0–100）：建议记录状态值与变化量，并与相关经济指标联动分析。
- 社会稳定/冲突指标（如 Rebellions）：建议记录次数、地点与是否成功（若有）。
- ...

## 6. 预期结果与限制

研究假设：【1–3 句，写清“变量之间可能的非线性关系/阈值效应”】【要求：不要写太多】

预期观察：【1 段话，说明在什么条件下可能出现什么现象（例如迁移集中、经济波动、冲突上升、财政约束收紧等）】

已知限制：【1 段话，说明时间窗/随机性/简化假设带来的影响与解释边界】
