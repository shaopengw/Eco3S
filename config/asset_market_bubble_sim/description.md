# 实验设计文档

## 1. 基本信息

**实验名称：** asset_market_bubble_sim

**实验目标：** 观察资产市场泡沫的形成与崩溃

**背景说明：** 创建持续双向拍卖市场，让智能体进行交易，探究资产价格偏离基础价值的泡沫与崩溃周期。

**原需求：** 创建一个持续双向拍卖市场。在该市场中，一项具有固定分红期限的虚拟资产（例如持续15个交易期的代币，每期固定分红0.1元，到期后价值归零）将被交易。智能体初始拥有不同的现金与资产组合，并基于其内部信念、对市场和其他智能体行为的观察进行买卖决策。系统需为每个智能体动态注入差异化的风险偏好、信息处理方式（如基本面分析、技术分析或跟风行为）以及私人信息（可能是有噪声的）。核心目标是通过完全去中心化的智能体交互，自下而上地涌现出资产价格偏离其基础价值的泡沫与崩溃周期。整个模拟环境需要记录并可视化所有交易、订单簿深度、个体信念与资产分布演变以及整体价格序列。

## 2. 系统模块选择

### 必选模块
- **居民系统** - 个体智能体的行为决策
- **地图系统** - 地理空间和城镇分布
- **时间系统** - 模拟时间步进管理
- **人口系统** - 人口增长、迁移、死亡

### 可选模块
- **政府系统** - 政策制定、税收管理
- **社交网络** - 人际关系、信息传播

## 3. 智能体设计

### 居民智能体
- **数量：** 100-500个
- **关键属性：** 现金、资产组合
- **核心行为：**
  - `buy_asset` - 购买资产
  - `sell_asset` - 出售资产
  - `analyze_market` - 分析市场走势
  - `follow_trend` - 跟随市场热点
  - `update_beliefs` - 更新个体信念

### 政府智能体 [可选]
- **核心职能：** 政策制定、监管市场

## 4. 环境设置

### 4.1 地理环境
- **城镇数量：** 1个
- **空间特征：** 集中分布
- **自然资源分布：** 无
- **城镇间距离：** 无

### 4.3 经济环境
- **基础生活成本：** 0

## 5. 关键指标

- **资产价格 (Asset Price)** - 观察资产价格随时间的变化
- **交易量 (Trading Volume)** - 记录每期的交易量
- **资产分布情况 (Asset Distribution)** - 统计每个智能体的资产分布情况

## 6. 模拟参数

### 6.1 基础参数
- **初始现金数量 (initial_cash)：** 100 units
- **分红金额 (dividend_amount)：** 0.1 units
- **交易期限 (trading_periods)：** 15 periods
- **模拟名称 (simulation_name)：** "asset_market_bubble_sim"

### 6.2 行为参数
- **响应概率 (response_probability)：** 0.1
- **最大重试次数 (max_retry_attempts)：** 3

## 7. 交互机制

- **居民 ↔ 居民：** 智能体之间进行买卖资产的交互
- **居民 ↔ 政府：** 政府监管市场，制定政策影响市场行为

## 8. 预期结果

**研究假设：** 智能体交互将导致资产价格出现泡沫并崩溃。

**预期发现：** 观察资产价格与基础价值的偏离程度，探究泡沫形成的时间和崩溃的因素。
