# 印度废钞令信息传播实验

## 1. 基本信息

本实验名称：**info_propagation（印度废钞令信息传播实验）**

本实验目标：比较不同信息长度与传播策略下，居民对废钞令关键规则的理解与选择差异（survey 类实验）。

背景设定：以 2016 年印度废钞令公告为核心刺激材料，居民在有限时间窗口内接收政策信息并形成理解与选择。信息来源可视为“官方告知”，同时允许社交网络影响个体的认知更新。

实验窗口与步进：总运行 3 个时间步（total_years=3），每步可视作一次信息接触与反应周期；不同条件之间重置居民状态以保证可比性。

## 2. 核心对象与实验设定

**参与者/智能体主体**：居民数量级为 200，每个居民具有以下基本属性：
- `gender`：性别
- `lifespan`：剩余寿命
- `satisfaction`：对目前生活的满意度 (0-100)
- `health_index`：健康状况 (1-5)
- `location`：位置 (x, y) 以及所属城镇 (town)
- `personality`：由随机形容词定义的性格

**行为**：
- `receive_and_decide_response`：接收官方消息，并立即规划是否发言。
- `spread_speech_in_network`：若决定发言，言论通过社交网络传播给所有相连邻居，触发对方记忆更新。
- `update_knowledge_memory`：在每一轮信息接收和讨论结束后，总结信息并更新长期知识记忆。

**刺激/信息与提示**：
刺激材料来自废钞令公告，包含短消息（`propaganda_message_short`）与长消息（`propaganda_message_long`）两种版本。长消息覆盖更完整的规则与限制条件，短消息保留核心条款。

## 3. 实验控制逻辑

本实验严格控制变量：同一批居民、同一套地图与社交结构、同一套评估问题，仅改变“信息长度/传播策略”作为自变量；观察理解度、选择分布与传播过程指标作为因变量。

**处理条件/实验组定义**（共四种策略，枚举 `PropagationStrategy`）：

1. **S_NCK（种子策略，无公共知识）**  
   - 根据综合中心度（度、中介、接近中心度加权平均）选取 `seed_count` 个种子居民，仅种子居民接收官方消息；非种子居民消息为空。  
   - 所有居民的 `public_notice` 均为“全村只有部分村民收到了政府信息。”

2. **S_CK（种子策略，有公共知识）**  
   - 种子选择与消息分发同上。  
   - 种子居民的 `public_notice` 为“全村只有部分村民收到政府信息，所有村民都得知你收到了政府信息。”；非种子居民的 `public_notice` 为“全村只有部分村民收到了政府信息。”

3. **BC_NCK（广播策略，无公共知识）**  
   - 官方消息发送给所有居民。  
   - `public_notice` 为空。

4. **BC_CK（广播策略，有公共知识）**  
   - 官方消息发送给所有居民。  
   - `public_notice` 为“你得知所有村民都收到了政府信息，并且所有村民都得知你收到了政府信息。”

## 4. 实验流程

**整体流程**：对每种处理条件依次运行，保证每个条件从相同的初始状态开始，记录过程与结果。

1. **初始化**：加载配置，创建居民、单一城镇及社交网络（异质图+超图）。
2. **策略循环**：依次执行四种传播策略（S_NCK, S_CK, BC_NCK, BC_CK）。对每种策略：
   1. **重置状态**：重置所有居民的记忆和实验相关计数器。
   2. **时间步循环**：运行 `total_years` 个时间步。在每个时间步中：
      - **信息呈现**：根据当前策略（广播/种子，CK/NCK）向居民分发官方消息及公共知识。
      - **个体反应**：居民接收信息，进行规划，并可能在社交网络中发言讨论。
      - **社交扩散**：发言内容通过社交网络传播，触发邻居的记忆更新。
      - **记忆总结**：所有居民调用 LLM 更新自己的长期知识记忆。
   3. **策略后评估**：时间步循环结束后：
      - **知识问卷调查**（`run_knowledge_survey`）：向所有居民发放标准化问卷，通过 LLM 解析答案，计算各题准确率和总体准确率。
      - **激励性选择问卷**（`run_incentive_survey`）：提出一个二选一的经济决策问题，统计选择分布，模拟对政策的真实信任程度。
   4. **保存结果**：记录该策略下的讨论量（`conversation_volume`）和问卷结果。

3. **完成与分析**：所有策略执行完毕后，将完整实验结果保存至 JSON 文件，并调用绘图功能生成对比图表，直观展示不同策略在讨论量、知识准确率和激励选择上的差异。

## 5. 关键指标

- **过程指标（Process）**：信息触达次数、社交传播轮次、响应率，以及**讨论量**（`conversation_volume`，即居民在社交网络中自发讨论政策的总次数）。
- **认知指标（Knowledge/Understanding）**：**知识问卷**的正确率（逐题准确率与总体准确率）。
- **行为/态度指标（Choice/Attitude）**：**激励性选择问卷**中支持/不支持或行动意愿的选择分布。

## 6. 可复现输出说明

- 运行参数：`initial_population=200`，`total_years=3`，`strategy=BC_CK`（示例），`response_probability=0.2`，`seed_count=1`。
- 结果输出：每个条件的统计结果与图表保存在 `history/info_propagation/` 对应实验目录下。
