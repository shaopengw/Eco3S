# “印度废钞令”信息传播实验（info_propagation）场景说明

## 1. 基本信息

本场景是一个偏“survey”的信息传播实验：核心目标不是让多个系统在长期互动中形成演化结果（那类更接近 decision 型/政策决策型的综合模拟），而是把环境与主体尽量固定下来，通过严格控制变量，比较不同信息传播机制对“讨论行为、知识掌握、激励性选择”的影响。

实验采用同一批居民、同一套社交网络与同一组问卷，在四种传播策略之间做对照。每种策略都会运行若干个时间步（以“天/轮次”为单位），策略结束后统一进行问卷评估，然后重置居民状态进入下一策略。这样可以把差异尽可能归因到“传播方式与公共知识条件”本身。

## 2. 核心对象与实验设定（自然语言口径）

居民是唯一的微观主体。每轮中居民会接收（或不接收）官方消息，并基于收到的信息与“公共知识提示”做出反应：可能在社交网络中发言讨论，也可能保持沉默；随后居民会更新自己的知识记忆，使信息在网络中逐步扩散。

实验中的“消息”来自配置文件，支持短消息/长消息两种版本（由 message_type 控制）；公共知识（common knowledge）并不改变消息内容，而是通过额外的 public_notice 改变居民对“他人是否也收到消息、他人是否知道我收到消息”的认知，从而影响其传播意愿与推理路径。

## 3. 实验控制逻辑（四策略对照）

实验枚举四种传播策略（PropagationStrategy），用两维因素交叉组成：

第一维：传播方式
- 广播（BC）：官方消息发送给所有居民。
- 种子（S）：只向少数“网络中心性更高”的居民发送官方消息，其余居民不接收消息内容。

第二维：公共知识条件
- CK（有公共知识）：通过 public_notice 提供“他人收到/知晓”的提示。
- NCK（无公共知识）：不提供或弱化上述提示。

四种策略分别为：S_NCK、S_CK、BC_NCK、BC_CK。

### 3.1 广播策略（BC）

广播策略下，所有居民都会收到同一条官方消息 message。
- BC_CK：额外附带 public_notice：
	“你得知所有村民都收到了政府信息，并且所有村民都得知你收到了政府信息。”
- BC_NCK：public_notice 为空。

### 3.2 种子策略（S）

种子策略下，会先在社交网络中选择 seed_count 个“种子居民”。选择标准是综合中心度排序：

综合评分 = 0.4 × 度中心度 + 0.3 × 中介中心度 + 0.3 × 接近中心度。

消息分发规则：
- 种子居民：收到官方消息 message。
- 非种子居民：不接收官方消息内容（message 为空）。

公共知识提示规则：
- S_CK：
	- 种子居民 public_notice：
		“全村只有部分村民收到政府信息，所有村民都得知你收到了政府信息。”
	- 非种子居民 public_notice：
		“全村只有部分村民收到了政府信息。”
- S_NCK：所有居民 public_notice 均为：
	“全村只有部分村民收到了政府信息。”

## 4. 实验流程（按策略循环）

实验总体按“策略”为外层循环：依次运行 S_NCK、S_CK、BC_NCK、BC_CK。每个策略内部包含三段：

1）重置状态：重置时间状态与居民的实验相关状态（包括记忆与计数器），确保每种策略从同一初始条件出发。

2）时间步循环：运行 total_years 个时间步。每个时间步大致包含：
- 执行策略：按当前策略构造 message 与 public_notice，并分发给居民；
- 居民反应：居民接收信息后决定是否发言；发言内容通过社交网络传播，触发他人更新；
- 记忆总结：所有居民更新长期知识记忆（短/长消息对应不同总结提示）。

3）策略后评估：时间步结束后，统一执行两类问卷调查：
- 知识问卷：检验居民对官方消息内容的理解与记忆；
- 激励性选择问卷：以二选一的经济决策题模拟对政策的真实信任与行为倾向。

最后保存该策略下的结果，并进入下一策略。

## 5. 关键指标（应包含与建议记录的数据）

本实验的评估单位是“策略”，因此建议把所有指标按策略分别汇总（必要时也可保留每个时间步的中间过程数据）。关键指标应至少包含：

- 讨论量（conversation_volume）：策略执行期间，居民在社交网络中自发讨论政策的总次数。它衡量“信息是否被带入公共讨论”。
- 知识问卷准确率（knowledge_survey）：
	- 每题准确率：每个问题正确回答的比例；
	- 总体准确率：所有居民×所有题目的整体正确率。
	这组指标衡量“信息是否被正确理解/记住”。
- 激励性选择分布（incentive_survey）：统计选择 A 与选择 B 的人数（或比例）。它用于衡量“即便知道信息，是否愿意做出更信任/更配合的行为选择”。

建议在策略对比时把三类指标放在同一张对照图或同一组表中：讨论是否更热、知识是否更准、选择是否更偏向某一行动。
