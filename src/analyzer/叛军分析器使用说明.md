# 叛军数据分析器 - 使用指南与说明文档

## 📋 概述

**叛军数据分析器（Rebellion Analyzer）** 是一个综合性数据分析工具，用于分析模拟中沿河和非沿河城镇的叛乱模式。它执行统计分析、趋势检测，并生成可视化图表，帮助理解地理位置（是否靠近运河）与叛乱频率之间的关系。

---

## 🎯 主要功能

### 1. **数据加载与解析**
- 读取叛军日志文件和城镇配置数据
- 自动将城镇分类为沿河和非沿河两组
- 从模拟日志中提取每年的叛乱决策记录

### 2. **统计分析**
- 每年叛乱频率统计（支持多次模拟的平均值和标准差）
- 描述性统计计算（均值、标准差、置信区间）
- 独立样本T检验（针对叛乱次数和叛乱率）
- 效应量分析（Cohen's d）
- 双重趋势分析：
  - 叛乱率差距随时间的变化趋势（年度层面）
  - 沿河城镇叛乱率的时间趋势（城镇-年份pooled regression）
- 跨实验比较指标（峰值强度、到峰时间、Gini系数等8项指标）

### 3. **可视化生成**
- 4种独立的分析图表（每个图表单独保存）
- 高分辨率PNG输出（300 DPI）
- 英文标签便于国际交流

### 4. **报告生成**
- 包含详细年度统计的CSV文件（含叛乱率、标准差等）
- 带有统计检验结果的综合文本报告（包含7个分析部分）
- 跨实验比较指标的JSON文件
- 结构化输出便于进一步分析和跨实验对比

## 🔬 计算方法和原理

### **1. 叛乱频率统计**

**方法：** 对于每一年，统计有多少城镇发生了叛乱（stage_rebellion > 0）

**公式：**
```
沿河城镇叛乱次数 = Σ (如果城镇在沿河城镇中 且 叛乱等级 > 0，则计1)
非沿河城镇叛乱次数 = Σ (如果城镇在非沿河城镇中 且 叛乱等级 > 0，则计1)
差距 = 沿河城镇叛乱次数 - 非沿河城镇叛乱次数
```

**目的：** 量化两类城镇之间叛乱活动的差异

---

### **2. 独立样本T检验**

**目的：** 确定沿河和非沿河城镇之间的叛乱频率差异是否具有统计学意义。

**原假设（H₀）：** 沿河和非沿河城镇的叛乱频率没有显著差异。

**备择假设（H₁）：** 沿河和非沿河城镇的叛乱频率存在显著差异。

**公式：**
```
t = (均值₁ - 均值₂) / √(方差₁/样本量₁ + 方差₂/样本量₂)

其中：
- 均值₁, 均值₂ = 各组的平均叛乱次数
- 方差₁, 方差₂ = 各组的方差
- 样本量₁, 样本量₂ = 样本大小（年份数量）
```

**解读方法：**
- **P值 < 0.05**：差异**具有统计学意义**（拒绝H₀）
- **P值 ≥ 0.05**：差异**不具有统计学意义**（不能拒绝H₀）

**示例结果：**
```
T统计量: 3.456
P值: 0.012
显著性 (α=0.05): 显著

→ 沿河城镇的叛乱次数显著多于非沿河城镇
```

---

### **3. 趋势分析（线性回归）**

**目的：** 确定沿河和非沿河城镇叛乱次数之间的差距是随时间增大还是缩小。

**公式：**
```
差距 = β₀ + β₁ × 年份 + ε

其中：
- β₀ = 截距（初始差距）
- β₁ = 斜率（每年的变化率）
- ε = 误差项
```

**关键指标：**

| 指标 | 含义 | 解读 |
|------|------|------|
| **斜率 (β₁)** | 每年的变化率 | 正值 = 差距扩大<br>负值 = 差距缩小 |
| **R²（决定系数）** | 拟合优度 | 0-1之间；越高越好<br>0.7以上表示强趋势 |
| **趋势P值** | 趋势的显著性 | < 0.05 = 趋势具有统计学意义 |

**示例结果：**
```
回归斜率: 0.23
R²: 0.68
趋势P值: 0.003

→ 差距每年增加0.23次叛乱
→ 该趋势解释了68%的变异
→ 趋势具有统计学意义
```

**示例：**

```
非沿河城镇标准差: 1.1

Cohens_d: 2.56
95%置信区间: [4.2, 6.4]
→ 效应量为大效应，差异的真实值有95%概率在4.2-6.4次之间
```

#### **第三部分：叛乱率统计（关键指标）**

**为什么叛乱率更重要？**
- 沿河和非沿河城镇数量可能不同
- 叛乱率 = 叛乱次数 / 城镇总数，消除了数量差异的影响
- 更科学地比较两组的叛乱倾向

| 指标 | 含义 | 关注要点 |
|------|------|----------|
| **平均叛乱率** | 每年每个城镇的平均叛乱概率 | 0.2表示平均20%的城镇叛乱 |
| **Cohens_d (效应量)** | 标准化的差异大小 | 量化实际影响的强度 |
| **95%置信区间** | 真实差异率的可能范围 | 更准确的差异估计 |

**示例：**
```
沿河城镇数量: 50个
非沿河城镇数量: 150个

沿河平均叛乱率: 0.17 (17%的城镇每年叛乱)
非沿河平均叛乱率: 0.02 (2%的城镇每年叛乱)
Cohens_d: 3.24 (大效应)
95%置信区间: [0.13, 0.17]
→ 沿河城镇的叛乱倾向是非沿河的8.5倍！
```

#### **第六部分：叛乱率差距趋势分析（核心分析）**

**重要说明：** 该部分使用叛乱率差距而非绝对次数差距，更科学地反映趋势。

| 指标 | 说明内容 | 示例解读 |
|------|----------|----------|
| **前半段平均叛乱率差距** | 早期年份的叛乱率差距 | 前半段: 0.15 |
| **后半段平均叛乱率差距** | 后期年份的叛乱率差距 | 后半段: 0.22 |
| **叛乱率差距变化** | 两段之间的差异 | 变化: +0.07 (扩大) |
| **回归斜率** | 每年叛乱率差距的变化 | 斜率: 0.0034/年 |
| **R²** | 趋势拟合数据的程度 | R² = 0.82 (拟合好) |
| **趋势P值** | 趋势是否显著 | P = 0.001 (是，显著) |

**完整示例：**
```
前半段平均叛乱率差距: 0.12
后半段平均叛乱率差距: 0.19
叛乱率差距变化: +0.07 (扩大)
回归斜率: 0.0035/年
R²: 0.79
趋势P值: 0.002

→ 叛乱率差距平均扩大了0.07 (7个百分点)
→ 叛乱率差距每年增加0.0035 (0.35个百分点)
→ 该趋势强劲(R² = 0.79)且具有统计学意义(p = 0.002)
→ 结论: 沿河城镇相对于非沿河城镇的叛乱优势随时间增强
```

#### **第七部分：沿河城镇叛乱率时间趋势分析（城镇-年份pooled regression）**

**分析目的：** 单独分析沿河城镇叛乱率随时间的演变，采用城镇-年份pooled regression方法。

**数据结构：**
- 对每个沿河城镇在每个时间步，计算跨多次模拟的平均叛乱率
- 汇总所有（城镇×时间步）的观测值进行回归分析
- 使用时间步（0, 1, 2, ...）而非实际年份（1650, 1651, ...）

| 指标 | 说明内容 | 示例解读 |
|------|----------|----------|
| **总观测数（城镇×年份）** | 回归分析的样本量 | 500个观测点 = 50个城镇 × 10年 |
| **沿河城镇数** | 参与分析的沿河城镇数量 | 50个 |
| **观测年份数** | 模拟的时间跨度 | 10年 |
| **前半段平均叛乱率** | 前半时期的平均叛乱率 | 0.15 |
| **后半段平均叛乱率** | 后半时期的平均叛乱率 | 0.22 |
| **叛乱率变化** | 两时期的差异 | +0.07 (上升) |
| **回归斜率** | 每个时间步叛乱率的变化量 | 0.0078/时间步 |
| **回归截距** | 初始时刻的平均叛乱率 | 0.142 |
| **标准误** | 斜率估计的标准误差 | 0.0012 |
| **R²** | 时间对叛乱率变异的解释程度 | 0.45 |
| **趋势P值** | 时间趋势的显著性 | 0.001 (显著) |

**完整示例：**
```
总观测数（城镇×年份）: 500
沿河城镇数: 50
观测年份数: 10

前半段平均叛乱率: 0.15
后半段平均叛乱率: 0.22
叛乱率变化: +0.07 (上升)

回归方程: rebellion_rate = 0.142 + 0.0078 × time_step
回归斜率: 0.0078/时间步
标准误: 0.0012
R²: 0.45
趋势P值: 0.001

→ 初始时刻沿河城镇的平均叛乱率为14.2%
→ 每个时间步叛乱率上升0.78个百分点
→ 10年间叛乱率累计上升约7个百分点
→ 时间解释了45%的叛乱率变异
→ 上升趋势具有高度统计学意义(p = 0.001)
→ 结论: 沿河城镇的叛乱风险随时间显著增加
```

**方法优势：**
- 利用城镇层面的数据，大幅增加样本量（城镇数×年份数）
- 提高统计检验的检验力（power）
- 能够捕捉沿河城镇群体层面的整体趋势
- 与差距分析互补，提供沿河城镇自身动态的独立视角

## 新增跨实验比较指标说明

为便于不同实验（不同随机种子/参数/干预）之间的可比性，分析器新增了一组直观指标，旨在从强度、时间、地理分布和稳定性多维度量化叛乱：

- **平均峰值强度（Mean Peak Intensity）**：每次模拟中年度叛乱城镇数序列的最大值的均值。反映在单次模拟中叛乱最严重的“峰值”规模；值越大表示峰值越强。

- **平均到峰时间（Mean Time To Peak）**：从模拟开始到达到峰值的平均年份（相对起始年）。用于衡量叛乱爆发的速度；值越小表示更早达到峰值。

- **城镇累积叛乱Gini（Gini of Cumulative Burden）**：对每个城镇在一次模拟中累积被袭次数（或被选为目标次数）取平均后，计算Gini系数以衡量叛乱负担的集中程度。Gini越接近0表示负担更均匀，越接近1表示高度集中于少数城镇。

- **城镇平均持续年数（Persistence）**：每次模拟中，城镇被记录为发生叛乱的年数的平均值，再对所有模拟取平均。该指标衡量叛乱在相同城镇上持续发生的程度，值越大表示同一城镇反复发生叛乱。

- **平均发生过叛乱的城镇比例（Prop Ever Rebel）**：每次模拟中至少发生一次叛乱的城镇数量占总城镇数的比例，取所有模拟的平均值。反映叛乱是否广泛分布或仅限少数城镇。

- **首发年(沿河 - 非沿河)平均差（First Rebel Year Difference）**：对每次模拟分别计算沿河城镇与非沿河城镇的首发年（首次发生叛乱的平均年份），再求两者差值并对多次模拟取平均。正值表示沿河城镇平均更早发生首发叛乱，负值则相反。

- **叛乱率差距AUC（AUC of Rate Gap）**：对叛乱率差距（沿河率 - 非沿河率）随时间序列计算积分（数值积分），表示差距在整个模拟期内的累计优势或劣势。正值表示沿河整体累积优势，绝对值越大表示累积差距越显著。

- **叛乱率差距波动（Gap Volatility, Std）**：叛乱率差距时间序列的标准差，衡量差距随时间的波动性。值越大表示差距不稳定、波动显著。

这些指标适合用于：

- 跨实验对比：对比不同参数/干预下的整体强度、集中度与时间动态；
- 归因与干预评估：若某种干预显著降低Gini或AUC，说明其有助于分散或抑制沿河的累积风险；
- 可视化补充：将这些指标与已有时间序列、箱线图结合，能更全面解释模型行为。

在 `rebellion_additional_metrics.json` 中会保存上述指标的数值，位于与log同目录，便于批量比较与统计汇总。

## 📚 统计学背景

### **为什么使用独立样本T检验？**
- 比较两个独立组的均值
- 检验观察到的差异是否可能由偶然发生
- 社会科学研究中广泛接受的方法
- 同时对叛乱次数和叛乱率进行检验，后者更科学

### **为什么使用线性回归进行趋势分析？**
- 量化随时间的变化率
- 提供趋势方向和强度
- 允许预测未来模式
- R²值衡量拟合优度

### **为什么使用城镇-年份pooled regression？**
- 利用城镇维度的信息，大幅增加样本量（城镇数×年份数）
- 提高统计检验的检验力（statistical power）
- 能够捕捉群体层面的整体趋势，而非仅依赖年度聚合数据
- 与差距分析互补，提供沿河城镇自身动态的独立视角

### **为什么计算叛乱率而非仅叛乱次数？**
- 消除城镇数量差异的影响（沿河和非沿河城镇数量可能不同）
- 更科学地反映叛乱倾向和风险水平
- 便于跨不同配置的实验进行比较
- 使结果更具可解释性和可比性

### **显著性水平 (α = 0.05)**
- 研究中的标准阈值
- 5%的假阳性概率
- 平衡I类和II类错误

### **效应量（Cohen's d）的重要性**
- P值仅告诉我们差异是否存在，不告诉差异有多大
- Cohen's d量化实际影响的强度
- |d| < 0.2：微小；0.2-0.5：小；0.5-0.8：中等；≥0.8：大
- 对于解释实际意义至关重要

## ❓ 常见问题（FAQ）

**Q1: 为什么要使用多个log文件？**
- A: 单次模拟可能受随机性影响，多次运行（3-5次）能提供更可靠的统计结果和标准差估计。

**Q2: 叛乱率差距趋势和城镇-年份趋势有什么区别？**
- A: 前者分析两组城镇之间差距的变化（年度层面），后者分析沿河城镇自身的时间演变（城镇×年份层面）。两者互补，提供不同维度的洞察。

**Q3: 额外指标（additional_metrics.json）有什么用？**
- A: 这些指标（如Gini、峰值强度、到峰时间等）专为跨实验比较设计，便于对比不同参数设置或干预措施的效果。

**Q4: 如果不同log文件的年份数不同怎么办？**
- A: 分析器会自动使用最小年份数进行对齐，确保所有模拟在相同时间跨度内比较。

**Q5: 如何解读R²值？**
- A: R²表示模型解释的变异比例。0.7以上表示强趋势，0.3-0.7中等，低于0.3表示趋势较弱或数据波动大。

**Q6: 为什么城镇-年份分析使用时间步（0, 1, 2...）而非实际年份？**
- A: 使用相对时间步便于不同起始年份的实验进行比较，且回归系数（斜率）的解释更直观（每步的变化率）。

**Q7: 统计报告中的"七个部分"都包含什么？**
- A: 1)城镇基本信息 2)叛乱次数统计 3)叛乱率统计 4)叛乱率差距趋势 5)叛乱次数差距趋势 6)沿河城镇叛乱率时间趋势 7)结论总结
- 提供趋势方向和强度
- 允许预测未来模式
